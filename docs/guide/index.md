# 介绍

ZoteroPatch AI Reader 是一个为 Zotero 7 PDF 阅读器设计的插件，通过集成 AI 功能，极大地提升文献阅读和笔记管理效率。

## 为什么选择 ZoteroPatch AI Reader？

### 🚀 无缝集成

插件深度集成到 Zotero 7 的 PDF 阅读器中，无需切换窗口或应用，所有 AI 功能触手可及。

### 💪 功能强大

- **智能翻译**: 支持多语言翻译，帮助阅读外文文献
- **自动摘要**: 快速生成论文摘要，把握核心内容
- **要点提取**: 结构化提取关键信息
- **智能问答**: 基于论文内容回答问题
- **笔记同步**: 无缝写入 Zotero 笔记系统

### 🔧 灵活配置

- 支持 OpenAI、国产大模型等多种 API
- 可使用本地部署的模型
- 丰富的自定义选项

### 🎯 专为学术设计

针对学术论文阅读场景优化，理解论文结构，提供精准的分析和总结。

## 核心功能

### 右侧栏 AI 面板

在 PDF 阅读器右侧栏显示 AI 助手面板，包含：

- **对话 Tab**: 进行问答交互
- **摘要 Tab**: 查看文档摘要
- **要点 Tab**: 查看提取的关键要点

### 选区快捷菜单

选中 PDF 文本后右键显示快捷菜单：

- 🌐 翻译
- 💡 解释
- ❓ 提问
- 📝 摘要

### 会话管理

- 每个文献独立的会话上下文
- 自动保存对话历史
- 智能管理内存占用

### 笔记同步

- 一键将 AI 内容写入 Zotero 笔记
- 支持追加和覆盖模式
- 自动添加时间戳和来源标记

## 技术特性

- **TypeScript** 开发，类型安全
- **模块化设计**，易于扩展
- **事件驱动架构**，响应迅速
- **资源友好**，内存和性能优化

## 系统要求

- Zotero 7.0 或更高版本
- 一个 OpenAI 兼容的 API Key
- 稳定的网络连接（使用云端 API 时）
